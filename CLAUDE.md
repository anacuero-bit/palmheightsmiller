# CLAUDE.md

## What This Is
Premium real estate and residency advisory website for high-net-worth individuals. Covers capital-efficient investments across UAE, USA, and Europe with golden visa structuring, tax optimization, and citizenship pathways.

## Markets
Dubai, Abu Dhabi, Ras Al Khaimah, Miami, New York, Athens, Lisbon, Madrid, Malta.

## Stack
- HTML/CSS/JS (static, single-file site in index.html)
- Google Fonts (DM Sans, Cormorant Garamond)
- Vanilla JS: market tab system, project carousel, scroll-reveal animations, Intersection Observer

## Hosting
- GitHub Pages

## Env Vars
None. Fully static site. Contact info has placeholder values marked with [XX].
